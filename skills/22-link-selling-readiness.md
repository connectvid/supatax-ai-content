# Skill 18: Link Selling Readiness
## What link buyers actually check. How to make every site sellable.

---

## The Gap Between SEO-Ready and Link-Sellable

A site can pass every technical SEO check and still be completely unsellable to
sophisticated link buyers. SEO readiness satisfies Google. Link selling readiness
satisfies humans with money who are skeptical by profession.

These are different audiences. Build for both.

---

## What Every Link Buyer Checks (In Order)

### Check 1: Ahrefs Traffic — Is It Real?

Link buyers open Ahrefs Site Explorer before anything else. They look for:
- Organic traffic trend: is it growing, flat, or declining?
- Traffic sources: is it all from one country? One keyword? (Red flag)
- Traffic velocity: did it spike overnight? (Signals manipulation)
- Top pages: are the high-traffic pages real tools/content? (Not redirected spam)

**What you need:** Organic traffic from multiple keywords across multiple pages,
growing gradually over time. A site with 1,000 visitors all from one "yes or no
wheel" keyword is less valuable than 1,000 visitors spread across 8 tools.

Build multiple tools before listing. The portfolio of traffic is more credible
than a single spike.

### Check 2: DR Legitimacy — Are the Links Real?

Sophisticated buyers check the backlink profile, not just the DR number.
Red flags they look for:
- 200 links appearing in one month (obvious Fiverr DR inflation)
- All links from the same IP range or registrar
- All links using exact-match anchor text
- Links from sites that are clearly spam (0-content, same template)

**What you need:** A mixed backlink profile. Directory links (Toolify, Product Hunt,
AlternativeTo) + social profiles (Twitter, LinkedIn) + 1–2 natural editorial links.
The Fiverr DR links should represent no more than 40% of total referring domains.
Spread the link building over 3+ months, not all at once.

### Check 3: Niche Relevance — Does Your Site Match Their Target?

A link from an SEO tool site (suparank.co) to an SEO client is worth $300.
A link from the same site to a dental practice is worth $0 — it's irrelevant.

**What you need:** Know which niches each site sells into and stick to them.

| Site | Primary Link Buyers |
|------|-------------------|
| ecomstudio.ai | Ecommerce tools, dropshipping courses, Amazon software |
| tweetsy.io | Social media tools, influencer platforms, entertainment |
| supatax.ai | Tax software, financial tools, payroll services, accounting |
| crazyads.io | Ad tech, marketing SaaS, creator economy tools |
| mailtoon.io | Email marketing tools, content tools, writing SaaS |
| maillead.io | Email verification services, outreach tools, B2B SaaS |
| suparank.co | SEO tools, content marketing, link building services |
| serpapis.com | SEO tools, web dev tools, API products |

Never sell links to clearly irrelevant niches — one irrelevant link sale tanks the
site's reputation in the link buyer community.

### Check 4: Contact Page — Does It Look Like a Real Site?

Link buyers email the contact page. If it looks like a generic form with no name,
no person, no email address — they move on.

**The contact page format that gets responses:**

```
Contact Us

For link insertion, sponsored posts, or editorial partnerships:
[Real email address — not a form-only page]
Response time: 24–48 hours

[Optional: brief description of what kinds of partnerships you accept]
```

Include a real email address (hello@domain.com or partnerships@domain.com).
The form is fine but the email address must be visible.
No name required — but if you use one, use a real-sounding name, not "Admin".

### Check 5: About Page — Does This Site Have an Identity?

Link buyers check if the site has a real reason to exist. A site with "We're a
team of passionate [niche] enthusiasts" is suspicious. A site that clearly explains
what it is and why it was built is trustworthy.

**The About page format that works:**

```
About [Site Name]

[Site name] is a free [tool type] platform built for [specific audience].
We provide [2–3 specific tools] to help [audience] [specific outcome].

All tools are free, no signup required.

[Optional: one sentence about why you built it — makes it human]
```

Do not: claim fake awards, fake team photos, fake "10 years of experience" claims.
Do: be specific about what the site does and who it's for.

### Check 6: Editorial Quality — Does Content Look AI-Generated at Scale?

Buyers who've been burned by AI link farms now scan content for mass-production
signals:
- Every page has identical structure down to the word count
- FAQ questions are obviously generic ("What is X?", "How does X work?")
- No site-specific personality or voice
- Homepage looks like a template with minimal customization

**What you need:** Variation in content length across pages (not all exactly 850
words). FAQ questions that reflect real search queries for that specific tool.
A homepage that speaks to the site's specific audience, not a generic "we provide
free tools" template.

### Check 7: Site Age — How Long Has It Been Indexed?

Most sophisticated buyers won't purchase from sites indexed less than 3 months.
The 3-month mark is when Google's sandbox effect typically lifts and traffic
becomes more stable.

**Minimum thresholds before listing:**
- 3 months since first indexing (not registration)
- 500+ monthly organic visitors on Ahrefs
- At least 5 tools live and working
- No penalty signals in GSC (no manual actions, no coverage issues)

## The Link Marketplace Setup

### Where to List

**Tier 1 (Start here):**
- Adsy.com — largest general marketplace, 30% commission
- Collaborator.pro — focused on editorial, 30% commission
- WhitePress — strong in Europe, 30% commission

**Tier 2 (Add after Tier 1 is generating revenue):**
- Direct outreach to SEO agencies (no commission, higher price)
- SEO Facebook groups (Link Building Mastermind, etc.)
- Twitter/X — posting Ahrefs screenshots publicly attracts buyers

### Pricing by DR + Traffic

| DR | Ahrefs Traffic | Price per Placement |
|----|---------------|-------------------|
| 20–30 | 500–1K | $50–80 |
| 30–40 | 1K–2K | $80–150 |
| 40–50 | 2K–5K | $150–250 |
| 50–60 | 5K–10K | $250–400 |
| 60+ | 10K+ | $400–600+ |

In commercial niches (SEO, finance, ecommerce): add 30% to these prices.
tweetsy.io (social/fun niche): subtract 20% — lower commercial value.

### Marketplace Profile Setup

For each site listing, you need:
- Site name and URL
- DR screenshot (from Ahrefs — take within 48 hours of listing)
- Ahrefs traffic screenshot
- Niche categories (select 2–3 relevant categories)
- Sample published posts showing content quality
- TAT (turnaround time): set to 3–5 business days
- Accepted content guidelines: set minimum 500 words, real content only

---

## The Pre-Listing Audit Checklist

Run this before listing any site on any marketplace:

**Traffic:**
- [ ] 500+ monthly organic visitors on Ahrefs
- [ ] Traffic from multiple keywords (not one viral page)
- [ ] Traffic growing or stable over 60+ days

**Authority:**
- [ ] DR 20+ (even with Fiverr boost)
- [ ] Mixed backlink profile (directories + social + editorial mix)
- [ ] No suspicious link velocity spikes in Ahrefs

**Content:**
- [ ] 5+ tools live and working correctly
- [ ] No placeholder content anywhere on site
- [ ] About page with real identity
- [ ] Contact page with real email address
- [ ] Privacy Policy page exists

**Technical:**
- [ ] No GSC manual actions
- [ ] No coverage issues (crawled not indexed must be < indexed)
- [ ] All pages return 200
- [ ] Site indexed for 90+ days

**Commercial:**
- [ ] Marketplace profile created with fresh screenshots
- [ ] Pricing set per the table above
- [ ] Niche categories correctly selected
- [ ] Response email monitored (check marketplace inbox daily)
